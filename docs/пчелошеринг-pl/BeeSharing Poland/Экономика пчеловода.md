# Экономика пчеловода-партнёра

## Что получает пчеловод
| Показатель | Значение |
|---|---|
| Закупочная цена мёда | 27 zł/кг |
| Для сравнения: скуп (hurtownia) | 18–22 zł/кг |
| Для сравнения: базар | 40–50 zł/кг |
| Гарантированный выкуп | ~115 кг при 8 ульях |
| Доход пчеловода за 8 ульев | ~3 100 zł |
| Хлопоты пчеловода | Ноль: ни фасовки, ни этикеток, ни маркетинга, ни доставки |

## Бонусы для пчеловода
- Бесплатные весы и камеры на его пасеке (он тоже видит данные)
- Его пасека на сайте с красивыми фото (реклама)
- Стабильный канал сбыта на годы
- Если у него 20–30 ульев, а мы берём 8 — остальные продаёт сам
- Доход от туров (50/50)
