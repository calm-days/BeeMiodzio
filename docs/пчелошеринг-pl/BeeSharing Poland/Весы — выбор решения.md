# Весы — выбор решения

## Ключевой инсайт

Достаточно **одних весов на всю пасеку** под контрольным ульем. Динамика привеса/убыли одинакова для всех ульев в одной локации. Так делает Пчелошеринг (Россия) — данные расшариваются всем клиентам в ЛК как общий индикатор медосбора.

> Для контрольного улья XS — лучший выбор: 400 кг ёмкость, лучшая точность (5г/°C), разница с 3W всего €40.

## Интеграция с нашим стеком

Наш стек: WiFi роутер на пасеке → VPS → InfluxDB → Grafana → ЛК клиента.

### Вариант A: BLE весы + ESP32 шлюз (рекомендация)

```
Весы (BLE) → ESP32 на пасеке (OpenMQTTGateway) → WiFi роутер → MQTT → InfluxDB → Grafana → ЛК
```

- BLE протокол BeehiveMonitoring [реверс-инжиниринг сделан](https://github.com/JoeRu/beehivemonitoring_devices)
- ESP32 декодирует через [Theengs Decoder](https://community.openmqttgateway.com/t/beehivemonitoring-com-devices-howto-add-support/1893)
- 0 подписок, полный контроль над данными
- Риск: ESP32 на улице может выйти из строя (автор оригинального проекта столкнулся с этим)

### Вариант B: GSM весы + HTTP API

```
Весы → GSM шлюз → облако beehivemonitoring → HTTP API → cron-скрипт → InfluxDB → Grafana → ЛК
```

API endpoint (недокументированный, но рабочий):
```
GET https://main.beehivemonitoring.com/api/hives/$hivenumber/history
Header: x-auth-token: <токен с QR-кода устройства>
Params: limit=1&reverse=true&attributes=weight;temperature;humidity
```

Данные: вес, температура, влажность, давление, напряжение батареи, счётчик пчёл, акселерометр.

- Надёжно, заводское решение
- Подписка: IoT SIM €16/год + web app (€19/год или €59 пожизненно)
- ⚠️ API недокументированный, токен привязан к веб-приложению — лицензия на app обязательна для работы API

### Вариант C: WiFi весы напрямую

```
Весы (WiFi) → наш роутер → облако производителя → (API?)
```

- BeePortal SCALE-1 — единственный вариант с прямым WiFi, но мало информации, непонятная цена
- Без своего API — те же проблемы с закрытой экосистемой

### Вариант D: DIY

```
ESP32 + HX711 + тензодатчики → WiFi роутер → MQTT → InfluxDB → Grafana → ЛК
```

- 220 zł, 0 подписок, полная интеграция
- Нужна сборка, калибровка, поддержка
- Прошивка: [Hiveeyes](https://www.hiveeyes.org/) (open source)
