# Ключевые риски и митигация

| Риск | Вероятность | Митигация |
|---|---|---|
| Слабый 4G в лесу | Средняя | Проверить до старта, направленная антенна +200 zł |
| Пчеловод отказался работать | Низкая | Контракт, 2–3 кандидата в воронке |
| Не продали 1/8 слоты | Средняя | Рождественский пик, Instagram, gift-платформы |
| Плохой урожай мёда | Низкая | Лесной мёд стабильнее моно-сортов; запасной пчеловод |
| Клиенты не продлили год 2 | Высокая (75% отток) | Модель работает на перепродаже слотов, не на удержании |
| Конкуренция появилась | Низкая (1–2 года) | First mover advantage, сеть пчеловодов как барьер |
| HoReCa: нет HACCP/badanie miodu | Средняя | Проверить до старта; часто пчеловод уже имеет от рыночных продаж |
| Choice paralysis (3 тира) | Низкая | Чёткие названия, визуальная дифференциация, Prezent как default |
| Выше COGS на подарочные наборы | Низкая | Компенсируется ростом среднего чека (+20% vs плоская цена) |
