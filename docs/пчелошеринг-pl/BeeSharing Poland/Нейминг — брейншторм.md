# Нейминг — брейншторм

## Фавориты

1. **BeeMine** — каламбур «be mine» + bee. Нежное, подарочное, работает на EU. Домены: beemine.pl / .co / .eu
2. **BeeSharing** — описательное, модель сразу в названии, паттерн carsharing. Домены: beesharing.pl / .co / .eu
3. **PocketHive** — улей в кармане, управляй с телефона. Геймификация + digital farm. Домены: pockethive.co / .pl
4. **HiveCraft** — Minecraft-ассоциация + крафт. Геймификация, не детская. Домены: hivecraft.co / .pl (hivecraft.com на продажу через Saw.com)
5. **BeeCraft** — то же, но фокус на пчелу. Домены: beecraft.pl / .co
6. **AdoptAHive** — «усынови улей». Эмоционально, понятна модель, подарочный потенциал. Домены: adoptahive.co / .pl
7. **BusyBeeClub** — клуб занятых пчёлок. Ирония + уютность + клубная модель. Домен: busybeeclub.co
8. **Pszczelа Chmurka** — «пчелиное облачко». Нежное, облачная пасека буквально. Домен: pszczelachmurka.pl
9. **Pasieczka** — «пасечка». Уменьшительное, тёплое, своё. Домен: pasieczka.pl
10. **Miodowa Chmurka** — «медовое облачко». Нежно + облачная пасека буквально. Домен: miodowachmurka.pl
11. **MójHive** — mój (мой) + hive. Польско-английская склейка, ownership. Домен: mojhive.pl
12. **MójUl** — «мой улей» по-польски. Просто, тепло, прямо. Домен: mojul.pl
13. **MyUlek** — my + ulek (улёк). Ласковое + международное. Домен: myulek.pl / .co
14. **BeeNasza** — bee + nasza (наша). «Наша пчела» — комьюнити. Домен: beenasza.pl
15. **BeeVillage** — деревня пчёл. Уютно, сообщество, визуально богато. Домен: beevillage.co
16. **BeeBliżej** — bee + bliżej (польск. «ближе»). «Будь ближе к пчёлам» — камеры, дашборд, присутствие. Каламбур: bee = быть + ближе. Домен: beebliżej.pl / beebliziej.pl
17. **PocketPasieka** — pocket + pasieka (польск. «пасека»). Пасека в кармане. PocketHive-вайб, но с польским словом — теплее и локальнее. Домен: pocketpasieka.pl / .co
18. **TwójUl** — twój (польск. «твой») + ul (улей). «Твой улей» — прямое обращение, ownership, интимнее чем MójUl (мой) потому что говорит клиенту «это твоё». Домен: twojul.pl
