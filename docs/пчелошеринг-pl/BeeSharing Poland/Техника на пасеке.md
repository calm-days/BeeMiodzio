### Макро-камера на леток — как это работает

Берём обычную IP-камеру с **варифокальным объективом** (2.8–12мм) и перенастраиваем фокус на минимальную дистанцию (~15–20 см). Камера ставится вплотную к летку. Так делает Пчелошеринг (Россия) — у них обычная корпоративная камера (Verkada), у которой перекрутили фокусное кольцо на макро.

Важные моменты:
- **ИК-подсветка отключена принудительно** — ИК беспокоит пчёл. Ночью экран чёрный, но пчёлы ночью не летают — наблюдать нечего. (Подтверждено опытом Пчелошеринга: «на прочих камерах ИК отключен принудительно, чтобы не беспокоить пчел»)
- **Питание по кабелю** от центрального бокса (PoE или 12V). Кабели прокладывать **в гофре** — иначе животные перегрызут (у Пчелошеринга собаки перегрызли незащищённый кабель)
- **Камера общего вида** (высоко, 10+ м) — можно оставить ИК для ночной картинки, на высоте пчёлам не мешает

## Весы — 1 контрольный улей (не на каждый)

Достаточно **одних весов на всю пасеку**. Подробный анализ всех решений: [[Весы — выбор решения]]

**Текущий выбор:** Комплект Intelligent Hive — **€377 (~1 640 zł).** Включает: весы XS (scientific, 400 кг, 10г) + 4G Solar GSM шлюз + Hive Heart 3.0 (датчик t°/влажности внутри улья) + батарейки + SIM (1 год включён). До 100 весов на 1 шлюз. Плюс lifetime лицензия на веб-приложение — €59 (нужна для работы API). **Итого: €436 (~1 900 zł).** Год 2+: только SIM €16/год (~70 zł).

Схема данных: Весы → GSM шлюз → облако beehivemonitoring → HTTP API → cron раз/час → InfluxDB → Grafana → ЛК
